
¿Por qué un AVL es mejor que un BST normal para ranking académico?
Como se pudo ver en clase, la razón de por que un Árbol AVL es mejor que un BST para
un ranking académico, porque garantiza que el árbol evolucione y cambie para ordenarlo y
balancearlo, cosa que con un BST es mera casualidad, por lo que es menos eficaz.
